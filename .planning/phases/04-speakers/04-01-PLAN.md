  <action>
Create `src/lib/speakers.ts` with these exact exports:

1. `getSlug(entryId: string): string` — strips locale prefix from collection entry ID. Implementation: `return entryId.replace(/^(fr|en)\//, "");`

2. `getLocale(entryId: string): Locale` — extracts locale from entry ID. Implementation: `return entryId.startsWith("en/") ? "en" : "fr";`

3. `getSpeakersByLocale(locale: Locale)` — calls `getCollection("speakers")` and filters by `s.id.startsWith(\`${locale}/\`)`. Returns the filtered array.

4. `getTalksByLocale(locale: Locale)` — calls `getCollection("talks")` and filters by `t.id.startsWith(\`${locale}/\`)`. Returns the filtered array.

5. `getTalksForSpeaker(locale: Locale, speakerSlug: string)` — calls `getTalksByLocale(locale)` and filters where `t.data.speaker === speakerSlug || t.data.cospeakers?.includes(speakerSlug)`. Returns matching talks.

6. `getSortedSpeakers(locale: Locale)` — calls `getSpeakersByLocale(locale)` and `getTalksByLocale(locale)`. Builds a `Set<string>` of keynote speaker slugs by iterating talks where `format === "keynote"` and adding both `talk.data.speaker` and each entry in `talk.data.cospeakers`. Sorts speakers: keynotes first (by `keynoteSpeakerSlugs.has(getSlug(s.id))`), then alphabetical by `s.data.name` using `localeCompare`. Returns sorted array.

7. `getCoSpeakersForTalk(talk: { data: { speaker: string; cospeakers?: string[] } }, currentSpeakerSlug: string): string[]` — combines `[talk.data.speaker, ...(talk.data.cospeakers ?? [])]` then filters out `currentSpeakerSlug`. Returns array of other speaker slugs.

Import `getCollection` from `"astro:content"` and `type Locale` from `"@/i18n/ui"`.

## Component Inventory

This phase introduces exactly one new visual component. All other work is infrastructure (routing, schemas, content pipeline).

### LanguageToggle

**Purpose:** Allows visitors to switch between French and English while preserving the current page.

**Placement:** Inside the top navigation bar (right side), visible on all pages. Renders in `Layout.astro` header area.

**Variant:** Ghost button pair (two adjacent ghost buttons in a segmented control style).

**Anatomy:**
```
[  FR  |  EN  ]
```

| Element | Specification |
|---------|---------------|
| Container | `--color-secondary` background, 1px `--color-border` border, `--radius-md` (6px), inline-flex |
| Container padding | 4px (xs) all sides |
| Button size | min-width 40px, height 32px (compact), 44px touch target via padding on mobile |
| Active button | `--color-primary` background, `--color-primary-foreground` text, `--radius-sm` (4px) |
| Inactive button | transparent background, `--color-muted-foreground` text |
| Label text | 14px, weight 500, uppercase, letter-spacing 0.05em |
| Hover (inactive) | `--color-secondary` background lightens slightly (increase OKLCH lightness 5%) |
| Focus | 2px `--color-ring` offset ring on the individual button |
| Transition | background-color 150ms ease |

**Interaction:**
- Clicking "EN" navigates to `/en/{current-path}` preserving the page
- Clicking "FR" navigates to `/{current-path}` (removes `/en/` prefix)
- Active state reflects the current locale from the URL
- The toggle is a navigation element (anchor tags), not a stateful toggle -- no client JS required for the switch itself

**Accessibility:**
- `aria-label="Language selector"` on the container
- `aria-current="true"` on the active language button
- Each button is a standard `<a>` element (keyboard accessible by default)
- Visible focus indicator via `--color-ring`

**Responsive:**
- Desktop: renders as described above, in the nav bar
- Mobile: renders in the mobile slide-in navigation panel, same dimensions

## Copywriting Contract

| Element | Copy (FR) | Copy (EN) |
|---------|-----------|-----------|
| Toggle label active | FR | EN |
| Toggle label inactive | EN | FR |
| Missing translation fallback heading | Contenu non disponible en francais | Content not available in English |
| Missing translation fallback body | Cette page n'a pas encore ete traduite. Consulter la version originale. | This page has not been translated yet. View the original version. |
| Missing translation fallback CTA | Voir en anglais | View in French |
| Schema validation build error | N/A (build-time only, developer-facing) | N/A |
| Empty collection state (dev) | Aucun contenu trouve dans cette collection. | No content found in this collection. |

**Notes:**
- No destructive actions in this phase
- No primary user-facing CTA -- the language toggle is the sole interactive element
- Missing translation fallback renders as an inline notice at the top of the page (not a full empty state), using `--color-card` background, `--color-muted-foreground` text, with a link styled in `--color-primary`

### Fallback Notice Component

| Element | Specification |
|---------|---------------|
| Container | full-width banner, `--color-card` background, 1px `--color-border` top/bottom border |
| Padding | 12px vertical (spacing-3), 16px horizontal (spacing-4) |
| Icon | `lucide:info` at 16px, `--color-primary`, inline before heading text |
| Heading | 14px, weight 500, `--color-foreground` |
| Body | 14px, weight 400, `--color-muted-foreground` |
| CTA link | 14px, weight 500, `--color-primary`, underline on hover |

## States & Interactions Summary

| State | Visual Treatment |
|-------|-----------------|
| French active | FR button shows primary background, EN button is ghost |
| English active | EN button shows primary background, FR button is ghost |
| Toggle hover (inactive button) | Subtle background lightening |
| Toggle focus | 2px ring in `--color-ring` |
| Missing translation | Inline banner with info icon, link to original locale |
| Invalid schema (build) | Astro build fails with Zod error -- no UI needed |
| Empty collection (dev) | Console/build warning only -- no runtime UI |

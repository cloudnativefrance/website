Define 4 collections:

**speakers** collection (glob loader, base: "./src/content/speakers", pattern: "**/*.md"):
Schema fields: name (z.string required), company (z.string optional), role (z.string optional), photo (z.string optional), bio (z.string required), social (z.object optional with twitter/linkedin/github/bluesky/website all z.string().url().optional())

**talks** collection (glob loader, base: "./src/content/talks", pattern: "**/*.md"):
Schema fields: title (z.string required), speaker (z.string required -- references speaker ID), cospeakers (z.array(z.string) optional), track (z.enum with values "cloud-infra", "devops-platform", "security", "community"), format (z.enum with values "talk", "lightning", "workshop", "keynote"), duration (z.number required -- minutes), room (z.string optional), startTime (z.string optional), tags (z.array(z.string) optional), youtubeUrl (z.string().url() optional), feedbackUrl (z.string().url() optional)

**sponsors** collection (file loader, path: "src/content/sponsors/sponsors.yaml"):
Schema fields: id (z.string required -- file() loader needs explicit id), name (z.string required), tier (z.enum with values "platinum", "gold", "silver", "community"), logo (z.string required), url (z.string().url() required), description (z.object with fr: z.string, en: z.string -- both required)

**team** collection (file loader, path: "src/content/team/team.yaml"):
Schema fields: id (z.string required -- file() loader needs explicit id), name (z.string required), role (z.object with fr: z.string, en: z.string -- both required), group (z.enum with values "core", "program-committee", "volunteers"), photo (z.string optional), social (z.object optional with twitter/linkedin/github/bluesky all z.string().url().optional())

Export as: `export const collections = { speakers, talks, sponsors, team };`

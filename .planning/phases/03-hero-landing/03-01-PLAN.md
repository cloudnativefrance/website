Create `src/components/hero/CountdownTimer.tsx` as a React island component (per D-03, D-04).

**Props interface:**
```typescript
interface CountdownTimerProps {
  lang: Locale;
}
```

**Core logic:**
- Target date: `new Date("2027-06-03T09:00:00+02:00").getTime()` stored as a module-level constant.
- `calcTimeLeft()` function: returns `{ days, hours, minutes, seconds }` or `null` if past the event.
- `useState<TimeLeft | null>(calcTimeLeft)` for initial state.
- `useEffect` with `setInterval(1000)` calling `setTimeLeft(calcTimeLeft())`. Always recalculate from `Date.now()` vs target (avoids drift per Research pitfall 2). Return cleanup `clearInterval`.

**Translation access:**
- Import `ui` and `Locale` from `@/i18n/ui`.
- Create local `t` helper: `const t = (key: string) => (ui[lang] as Record<string, string>)[key] ?? key;`

**Post-event mode (D-04):**
- When `timeLeft` is `null`, render a post-event section:
  - Text: `t("hero.post_event")` in `text-xl font-bold text-foreground`
  - Button: `<a href="#replays">` styled as primary button with `t("hero.cta.replays")` text, same glow shadow as register CTA.

**Pre-event countdown cards (D-03):**
- Container: `<div role="timer">` with flex row, `gap-3 sm:gap-4`, centered.
- For each unit (days, hours, minutes, seconds), render a digit card:
  - Card wrapper: `bg-card border border-border rounded-lg min-w-[60px] sm:min-w-[72px] py-3 px-2 text-center`
  - Value: `<span className="block text-xl sm:text-5xl font-bold text-foreground" style={{ lineHeight: '1.1', letterSpacing: '-0.02em' }}>` with value zero-padded to 2 digits (use `String(val).padStart(2, '0')`).
  - Label: `<span className="block text-base font-normal uppercase tracking-widest text-muted-foreground mt-1">` with `t("countdown.days")` etc.

**Accessibility (per UI-SPEC):**
- `role="timer"` on container.
- `aria-label` with full time remaining text, e.g., `${days} ${t("countdown.days")}, ${hours} ${t("countdown.hours")} remaining`.
- Throttle `aria-label` updates to every 60 seconds using a `useRef` for last update timestamp. The visual display still updates every second.

**Export:** `export default function CountdownTimer({ lang }: CountdownTimerProps)`

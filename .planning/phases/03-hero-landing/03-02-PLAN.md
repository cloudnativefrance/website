**Translation access:**

**useCountUp hook (internal to this file):**
```typescript
function useCountUp(target: number, duration: number, start: boolean): number
```
- Uses `useState(0)`, `useEffect` with `requestAnimationFrame`.
- Ease-out cubic: `1 - Math.pow(1 - progress, 3)` where `progress = Math.min(elapsed / duration, 1)`.
- Duration: 2000ms.
- Returns `Math.floor(eased * target)`.
- Cleanup: `cancelAnimationFrame`.

**IntersectionObserver trigger:**
- `useRef<HTMLDivElement>(null)` on the container.
- `useState(false)` for `visible`.
- `useEffect` sets up `IntersectionObserver` with `{ threshold: 0.3 }`.
- When `isIntersecting`, set `visible(true)` and `observer.disconnect()` (animate only once).
- Cleanup: `observer.disconnect()`.

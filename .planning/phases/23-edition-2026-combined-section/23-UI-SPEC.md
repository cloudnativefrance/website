Primary (Blue) is reserved for both inline CTAs.

**Accent reserved for:** Hero "Restez informé" newsletter CTA (Phase 25 only) and the global past-editions PLACEHOLDER badge (`bg-accent text-accent-foreground` — `PastEditionSection.astro` line 61). Do **not** apply `bg-accent` / `border-accent` / `text-accent` to anything in `Edition2026Combined.astro`.

**Contrast notes:**
- Quote text on `--color-card`: `--color-card-foreground` is the same lightness as `--color-foreground` (95.6%) over a 22.5% surface — exceeds WCAG AA 4.5:1 by a wide margin.
- CTA primary text on background: oklch 62.5% on 16.8% — meets WCAG AA for non-text and large UI components; the `hover:underline` provides redundant affordance.
- Caption / attribution muted-foreground (66.8%) on background (16.8%): `~6.2:1` — passes AA.

### Block-by-block contract

**1. Rail (decorative, optional):** Same pattern as `PastEditionSection.astro` lines 75–80. Rotated `-90deg` rail, `hidden md:block`, `text-muted-foreground`, `tracking-[0.2em]`. Reuses existing key `editions.rail.2026` — no new copy. Decision: KEEP the rail to preserve vertical rhythm with `PastEditionMinimal` (which has no rail) and the rest of the homepage past-editions strip.

**2. Heading h2:** Verbatim style from `PastEditionSection.astro` line 83. Single h2 in this section (Pitfall 8 mitigation). Reuses existing key.

**3. Photo mosaic:** 3 thumbnails (per D-04). Grid spec verbatim from `PastEditionSection.astro` lines 99–112. The `colSpan` map is:
```ts
const colSpan = {
  hero:   "col-span-12 md:col-span-6",
  medium: "col-span-6  md:col-span-3",
  small:  "col-span-6  md:col-span-3",
} as const;
```
**Photo size assignments (UI-SPEC decision, supersedes editions-data.ts current `size: "hero"` for all 4):**
| Slot | File | Size | Rationale |
|------|------|------|-----------|
| 1 | `ambiance-03` | `hero` | Anchor image — shows main hall ambiance, strongest visual |
| 2 | `ambiance-06` | `medium` | Networking moment — secondary |
| 3 | `ambiance-10` | `medium` | Overall venue view — closes the row |
**Drop:** `ambiance-08` (per D-04). Update `EDITION_2026.thumbnails` in `editions-data.ts` to the 3-element ordered array above with the new `size` values. Renders as one row (`[grid-auto-rows:12rem] md:[grid-auto-rows:14rem]`) — 1 wide-left + 2 stacked-equivalent right on desktop.

**4. Video embed:** Verbatim YouTube pattern from `PastEditionSection.astro` lines 116–127, minus the `playlistCta` (CTAs live in block 5 instead). `youtube-nocookie.com/embed/${EDITION_2026.youtubeId}`, `aspect-video`, `loading="lazy"`, `allowfullscreen`, container `max-w-2xl mx-auto`.

**5. CTA row:** Two siblings, one `<a>` each. Style is the gallery-cta pattern from `PastEditionSection.astro` lines 152–162 — `inline-flex items-center gap-2 text-sm font-semibold text-primary hover:underline` + trailing `<span aria-hidden="true">→</span>`. **Both** anchors carry `target="_blank" rel="noopener noreferrer"` (D-11). The PDF CTA additionally carries `aria-label={t("editions.2026.pdf_cta_aria")}` so screen-reader users hear "(PDF, new window)" — Pitfall 6 mitigation. On mobile: `flex-col gap-4` (stacked). On `sm+`: `flex-row gap-8` (side by side, left-aligned, NOT centered). Container `max-w-2xl mx-auto` so they sit visually under the video.

**6. Testimonials block:** Single h3 followed by a 3-up grid. Card style derived from `TestimonialsStrip.astro` lines 83–101 (border + radius + background + italic blockquote + muted attribution) **without** the marquee CSS (`@keyframes scroll-x`, mask, `animation-play-state`). Static cards. No avatars (per anti-features in FEATURES.md). No hover transform. The `prefers-reduced-motion` global reset (`global.css` lines 103–112) covers the entire component for free.

| Viewport | Photo mosaic | Video | CTA row | Testimonials grid |

| `≥768px` (md) | One row: 1 hero (md:col-span-6) + 2 medium (md:col-span-3 each) side by side | `max-w-2xl` centered | Single row, left-aligned, `gap-8` | 3 columns, `gap-8` |

### Interaction states

| Element | Default | Hover | Focus (keyboard) | Disabled |
|---------|---------|-------|------------------|----------|
| Replays CTA | `text-primary` | `underline` | `outline-ring/50` (global `*` rule in `global.css` line 91) | n/a |

import type { ImageMetadata } from "astro";

export interface Props {
  /** Anchor id for deep-linking. Recommended: "edition-2026". */
  id?: string;
  /** Localised rail label, e.g. t("editions.rail.2026"). */

  heading: string;

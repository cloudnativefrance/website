# GSD planning archive

This directory is the frozen state of the project's **GSD** (get-shit-done) planning artifacts at the moment of migration to [Superpowers](https://github.com/obra/superpowers).

Contents are kept for historical reference only. Do not add new planning here — use the Superpowers workflow instead.

Previous layout:
- `PROJECT.md`, `ROADMAP.md`, `STATE.md` — top-level project state
- `milestones/` — per-milestone roadmaps, requirements, and audits (v1.0 → v1.2)
- `phases/` — per-phase plans, research, reviews, and verifications (phases 23–26)
- `workstreams/` — parallel workstream tracking (past-editions-showcase)
- `research/` — codebase research snapshots
- `config.json` — GSD runtime config

- **24-01:** safeUrl + safeLogoPath extracted to `src/lib/sponsor-utils.ts` (named exports); SocialLinks.astro keeps its own inline copy (cross-component extraction deferred to v1.3); `sponsors.homepage.cta` uses Pattern B (no arrow in i18n value, decorative `<span aria-hidden="true">→</span>` in template)
- **24-02:** SponsorsPlatinumStrip data-agnostic (caller owns `getCollection("sponsors")` + tier filter); belt-and-braces empty-state guard; no SponsorCard nesting (local cardClasses string instead — homepage strip diverges from Platinum tier card)
- **24-03:** Edition2023Link fully prop-driven (no imports from `@/lib/editions-data` or `@/i18n/utils`); Pattern A arrow on /2023 CTA (glyph in i18n value); h2 uses `font-semibold + tracking-tight` (one notch lighter than Sponsors strip); logo on bare background
- **25-01:** Background opacity locked to single `<img>` @ `opacity-75` (no overlay div); contrast math passes AA over existing GeoBackground scaffolding; mail icon hand-inlined SVG (NO `lucide-react` import added — mirrors site-wide convention); newsletter placeholder anchor `#newsletter-stub` initially → swapped to real SibForms URL post-UAT; Ghost CTA focus ring overrides default `ring-ring/50` to `ring-accent/50`; mobile button widths intrinsic (NOT `w-full`)
- **26-01:** Sponsor tier filter is lowercase `"platinum"` (matches Zod schema `z.enum(["platinum", ...])`); locale-aware hrefs hardcoded as string literals (`/2023` vs `/en/2023`, `/sponsors` vs `/en/sponsors`); orphan `editions.2026.gallery_cta` reference removed (fixes 2 of 11 baseline astro-check errors)
- **26-02:** Favicon SVG-only swap accepted; modern browsers (≥99% traffic) get new flag immediately via existing `<link rel="icon" type="image/svg+xml">` declaration; `favicon.ico` initially kept as legacy fallback, then deleted in post-UAT
- **26-03:** Cleanup grep gate scoped to `src/` only — missed `tests/` consumers; 4 broken/stale tests fixed in `73fdfa1` follow-up commit; PastEditionSection's testimonials data file (`testimonials-data.ts`) preserved (still consumed by Edition2026Combined)
- **Post-UAT:** Promoted KubeCorp `gold → platinum` to deliver 3 Platinum sponsors; switched grid layout to `flex flex-wrap justify-center`; replaced `#newsletter-stub` placeholder with real SibForms URL; revised PDF CTA copy (`pdf_cta` + `pdf_cta_aria`) to "Présentation de l'événement (OnePager PDF)" / "Event overview (OnePager PDF)"; added Ente photo gallery CTA in Edition 2026; removed "Événement officiel KubeCon Community Days" footer affiliation; added `?v=2` cache-bust on favicon SVG link
